# 🛒 Elektro Ventas

Elektro Ventas es el proyecto final del curso de React en Talento Tech.

[![Demo](https://img.shields.io/badge/Demo-Live-success)](https://elektro-ventas.vercel.app)
[![React](https://img.shields.io/badge/React-19.1.1-blue)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-7.1.7-646CFF)](https://vitejs.dev/)

## 🌟 Descripción

Es una plataforma de ventas en línea que ofrece una experiencia de usuario fluida y moderna. El proyecto está construido con las últimas tecnologías de desarrollo web frontend.

## 🚀 Demo

Visita la aplicación en vivo: [https://elektro-ventas.vercel.app](https://elektro-ventas.vercel.app)

## ✨ Características

- 🎨 Interfaz de usuario moderna y responsiva
- ⚡ Rendimiento optimizado con Vite
- 🔄 Navegación fluida con React Router
- 📱 Diseño adaptable a diferentes dispositivos
- 🎯 Gestión de estado con Context API
- 🔒 Login para CRUD de datos

## 🛠️ Tecnologías

- **Frontend Framework:** React 19.1.1
- **Build Tool:** Vite 7.1.7
- **Routing:** React Router DOM 7.9.4
- **Lenguaje:** JavaScript (ES6+)
- **Linting:** ESLint 9.36.0
- **Deployment:** Vercel

## 📋 Requisitos Previos

Antes de comenzar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión 16 o superior)
- npm o yarn

## 🔧 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/SebasTerco73/Pagina-ventas.git
```

2. Navega al directorio del proyecto: 
```bash
cd Pagina-ventas
```

3. Instala las dependencias:
```bash
npm install
```

## 💻 Uso

### Modo Desarrollo

Ejecuta la aplicación en modo desarrollo: 

```bash
npm run dev
```

La aplicación se abrirá en [http://localhost:5173](http://localhost:5173)

### Build para Producción

Genera una versión optimizada para producción:

```bash
npm run build
```

### Preview de Producción

Previsualiza la build de producción localmente:

```bash
npm run preview
```

### Linting

Ejecuta el linter para revisar el código:

```bash
npm run lint
```

## 🚀 Deployment

El proyecto está configurado para desplegarse automáticamente en Vercel.  Cada push a la rama `main` despliega una nueva versión. 

### Desplegar tu propia versión

1. Haz fork del repositorio
2. Conecta tu repositorio con [Vercel](https://vercel.com)
3. Configura el proyecto (Vercel detectará automáticamente Vite)
4. ¡Listo! Tu aplicación estará en línea
